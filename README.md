# ClickyTvRemote
A custom IR TV remote with mechanical keyboard switches

The project started because we have no remote any more for our Philips 32PFL5403D/12 TV.

IR codes for this Model can be found in the [IRDB on Github](https://github.com/probonopd/irdb/blob/master/codes/Philips/Unknown_32PFL5403D/0%2C-1.csv).
